Detailed Description

Creates and moves nodes controlled by a TraCI server.

If the server is a SUMO road traffic simulation, you can use the TraCIScenarioManagerLaunchd module and sumo-launchd.py script instead.

All nodes created thus must have a TraCIMobility submodule.

See the Veins website for a tutorial, documentation, and publications .

Author
